## About The Project

採用瀑布流（Masonry Layout）的視覺設計，重新排版巴哈姆特場外休憩區的文章</p>
本專案為實戰自學之side project</p>
爬蟲所呈現的資料皆不會在重整後持續存在，應用程式內無插入廣告，並且開源
